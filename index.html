Weather App

This is a responsive weather application that fetches real time weather data for any city using the weather API from SheCodes. Users can view temperature, humidity, and weather conditions from their location or any other location. It was made using HTML, CSS, and JavaScript.

World Clock

This is a world clock application, that displays the current time in multiple cities around the globe. It was built with HTML, CSS, and moment.js, it allows users to track time zone with ease.

AI Crochet Pattern Generator

An AI-generator for crochet artists or beginner crochet artists, that generates custom crochet patterns based on user input. Users can specify the item they want, and it'll generate a beginner friendly pattern for them, with the yarn type, yarn weight and hook size. It was built with HTML, CSS, JavaScript, and the SheCodes AI API.
